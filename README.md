# VRM Backend Core API

This repository contains the **backend core API foundation** for a Vendor Risk Management (VRM / TPRM) platform.

The goal of this backend is to provide a **secure, modular, and auditable** API layer that supports:
- Role-based access control (RBAC)
- Organization-level tenant isolation
- Audit-grade activity logging
- Clean REST APIs for frontend and QA integration
